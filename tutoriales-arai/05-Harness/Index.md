# Creación de Harness

Un **harness** en opencode es la configuración completa que define el runtime, agentes, skills, permisos, comandos y herramientas de un proyecto. Esta sección cubre el ecosistema completo de creación de harness.

3. [[05-Harness/03-harness-generator.md|Harness Generator]] — genera un harness completo desde una descripción JSON.
4. [[05-Harness/04-ciclo-completo.md|Ciclo completo de creación]] — desde la idea hasta el harness funcionando.
