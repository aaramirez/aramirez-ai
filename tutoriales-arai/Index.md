# Tutoriales arai

> Guías paso a paso para usar **arai** — el configurador multi-agente de opencode.

## Contenido

### [[01-Instalacion/Index|Instalación]]
Cómo instalar arai, preparar el entorno y crear tu primer proyecto con opencode.

### [[02-Comandos/Index|Comandos esenciales]]
Todos los comandos de arai explicados con ejemplos: init, install, uninstall, sync, status, generate, list, update.

### [[03-Configuracion/Index|Configuración de opencode]]
El archivo `opencode.json` a fondo: modelo, runtime, permisos, shell, formateadores, LSP.

### [[04-Agentes/Index|Arquitectura de agentes]]
Agentes primarios, subagentes, agentes especializados, arquitecturas multi-agente y flujos de trabajo.

### [[05-Harness/Index|Creación de Harness]]
El ecosistema de creación de harness: 18 creator scripts, `harness-generator`, y cómo generar configuraciones completas.

### [[06-Skills/Index|Trabajar con skills]]
Usar skills existentes, crear skills nuevas, sincronizar y compartir skills entre proyectos.

### [[07-MCP/Index|MCP, Comandos y Extensiones]]
Servidores MCP, comandos personalizados, plugins y herramientas personalizadas.

### [[08-Referencias/Index|Prompts, Reglas y Branding]]
Fragmentos de prompt, reglas de codificación, referencias compartidas, branding y repositorios de referencia.

### [[09-Documentacion/Index|Generación de documentos]]
El pipeline docgen: decks, reportes, imágenes, PDFs desde especificaciones JSON y exportación de vaults.

### [[10-CI/Index|Validación y Calidad]]
CI validation, tests, docgen validate y outcome validation.

### [[11-Casos-de-uso/Index|Casos de uso]]
Escenarios completos que integran múltiples capacidades de arai.

---

> **Nota**: Todos los tutoriales asumen que tienes Node.js 18+ y Git instalados.
