---
description: Send an email using the project SMTP configuration.
---

1. Understand the email requirements from the user: to, subject, body, cc (optional), bcc (optional), html (optional), attachments (optional — file paths to attach).
2. If the user wants to send a report or document generated by docgen, first run the appropriate `npm run docgen:*` command to generate it, then include the output path as an attachment.
3. Build a JSON payload with those fields and write it to `.opencode/email-payload.json`.
4. Execute: `node shared/scripts/send-email.js --payload .opencode/email-payload.json`
5. Report success (include the messageId) or error details.
