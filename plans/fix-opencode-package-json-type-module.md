# Fix: Ensure `.opencode/package.json` includes `"type": "module"`

## Objective

Ensure that when a new harness is generated via `arai init` or `arai install`, the `.opencode/package.json` file includes `"type": "module"` so that ES module scripts like `getrepo.js` run without the `MODULE_TYPELESS_PACKAGE_JSON` warning.

## Problem

- `.opencode/package.json` is auto-generated by **opencode** (not arai) when it detects plugins
- opencode only adds `@opencode-ai/plugin` dependency — no `"type": "module"`
- Scripts in `.opencode/scripts/` (like `getrepo.js`) use ES module syntax (`import`/`export`)
- Without `"type": "module"`, Node.js emits a warning and re-parses as ESM (performance overhead)

## Requirements

1. New harnesses must have `"type": "module"` in `.opencode/package.json` — priority: high
2. Existing installations should be patched on next `arai install` — priority: high
3. No breaking changes to existing behavior — priority: high
4. Cross-platform: Node.js only — priority: medium

### Decisions

1. **Where to add the logic**: In `install.js` as a reusable function, called from both `installPlatform()` and `scaffoldProject()`
2. **Strategy**: Read existing `.opencode/package.json` if present, merge `"type": "module"` in; if missing, create minimal file with just `"type": "module"`
3. **Idempotent**: Safe to call multiple times — only adds `"type": "module"` if not already present

## Implementation

### Step 1: Add helper function in `install.js`

```js
function ensureOpenCodePackageJson(projectRoot) {
  const pkgPath = join(projectRoot, '.opencode', 'package.json');
  let pkg = {};

  if (existsSync(pkgPath)) {
    try {
      pkg = JSON.parse(readFileSync(pkgPath, 'utf8'));
    } catch {
      log('Could not parse .opencode/package.json, recreating', 'warn');
    }
  }

  if (pkg.type === 'module') return; // already correct

  pkg.type = 'module';
  ensureDir(dirname(pkgPath));
  writeFileSync(pkgPath, JSON.stringify(pkg, null, 2) + '\n');
  log('Ensured .opencode/package.json has "type": "module"', 'ok');
}
```

### Step 2: Call from `installPlatform()`

After creating `.opencode/` directory (line 70), call `ensureOpenCodePackageJson(projectRoot)`.

### Step 3: Call from `scaffoldProject()`

After scaffolding `.opencode/` contents (around line 118), call `ensureOpenCodePackageJson(absTarget)`.

### Step 4: Export the function

Add `ensureOpenCodePackageJson` to the exports for testing.
